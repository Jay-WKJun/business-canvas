# Antd 테이블

Antd를 활용한 테이블입니다.

# 실행방법

```bash
# pnpm이 없는 경우
$ npm i -g pnpm

$ pnpm install

$ pnpm dev

# 브라우저에서 http://localhost:5173/ 에 접근
```

## 테스트 실행

```bash
$ pnpm test
```
